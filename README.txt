1. Для работы проекта используется PostgreSQL.
2. В качестве контейнера сервлетов был выбран apache tomcat9. Вариант, используемый при создании
проекта - использовать архивную версию в сочетании с IDE Intellij Idea.
3. Файл настройки находится в директории Cinema/src/main/resources.
4. Для запуска из Idea - добавить tomcat в конфигурацию запуска и выбрать тип артефакта exploded.
